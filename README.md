# Insert-Data-in-File-using-Flask
A simple Flask web application that allows users to enter text through a form and saves the submitted data into a text file on the server. Perfect for beginners learning Flask, file handling, and basic routing.

#How to Run the Project
*Step 1: Install Flask
pip install flask

*Step 2: Run the Flask server
python app.py

*Step 3: Open in browser
http://127.0.0.1:5000/

#Example Input Flow

User enters text in the form
Clicks Submit
Text is saved into text.txt
Page reloads automatically
